# landing-preview — design source (not the live page)

This folder holds the original design exploration for the Bizra Farms front
door. It is kept as **design reference only**.

**The live page is `apps/frontend/public/bizra.html`** (served at `/`, with
photos in `apps/frontend/public/bizra-assets/`). Editing files here does **not**
change the deployed site — make front-door changes in `public/bizra.html`.

Front-door flow:  `/` (bizra.html)  →  `/landing` (landing.html)  →  `/dashboard`.

- `bizra-economicbridge/index.html` — the initial combined Bizra + EconomicBridge
  draft this page was built from.
- `bizra-economicbridge/bizra-logo.jpg` — logo (the production copy lives at
  `apps/frontend/public/bizra-assets/bizra-logo.jpg`).
